## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/Muskan0018/My-Portfolio.git
```

Navigate to project folder:

```
cd My-Portfolio
```

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

---

## 📦 Build for Production

```
npm run build
```
